โปรเจค PCI card ตอนที่ 2 Aries Board Version A.1

1. บอร์ด

ในที่สุดก็ออกมาเป็น version A.1 แล้วครับ กับ Aries board ที่ออกแบบมาให้ใช้งานได้หลากหลาย เพื่อที่จะได้ไม่ต้องทำบอร์ดใหม่ทุกครั้งที่อยากจะทดลองอะไรใหม่ๆ

2. Board specification

spec ของบอร์ดมีดังนี้ครับ

Tile Specification Note
FPGA Cyclone II 5000 LE (EP2C5Q208C8)
Memory SDRAM: 256Mbit
FPGA config ROM: 4Mbit
Serial Flash memory: 4Mbit
PCI interface 32 bits 66MHz/33MHz Default: 33Mhz
ADC 12 bits / 8 Channels (MCP3208 x 1)
DAC 12 bits / 4 Channels (MCP4922 x 2)
Digital I/O 16 Channels
RS232 1 Channel
LED 8 bit Share with VGA
VGA 8 bit (RGB:222) Share with LED
Swtich Tact SW x3
Slide SW x 4
Input power 5V

ส่วนของ USB connector ที่มีบนบอร์ดนั้น จะทำงานร่วมกับ USB blaster clone ที่เอาไว้ program firmware ผ่านทาง JTAG ครับ http://sa89a.net/mp.cgi/ele/ub.htm

ตอนนี้ทดลอง config FPGA ด้วย NIOS2 แล้ว run test program เพื่อทดสอบการ์ด PCI ได้แล้ว
